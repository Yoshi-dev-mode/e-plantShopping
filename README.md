# e-plantShopping

## Project Overview

e-plantShopping is a React-based web application that allows users to browse and purchase plants online. The application provides an easy-to-use interface for viewing plant collections, adding plants to a shopping cart, and managing purchases.

## Features

- Browse a variety of plants
- Add plants to the shopping cart
- View and manage cart items
- Responsive and user-friendly interface

## Technologies Used

- React
- JavaScript
- HTML
- CSS
- Redux Toolkit
